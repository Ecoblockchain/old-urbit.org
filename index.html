Last Continuity Breach: December 20th, 2013

A continuity breach occurs when the entire Urbit network gets recreated (usually for development reasons). All ships and pier prior to the breach will no longer work with the new network.

What does this mean for the Urbit user? If you built your ships and piers before December 20th, you need to run the following:

  1. git pull origin master
  2. make clean; make
  3. rm -rf $URBIT_HOME/$ships (except for /zod and urbit.pill)
  4. rm -rf $piers
  5. vere -c $mypier and create your submarine/destroyer
  6. :chat to chat
